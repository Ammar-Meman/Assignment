HTML Elements Showcase - Inline vs Block

In HTML, elements are split into two main groups: **block** and **inline**. It's a key concept to understand because it controls how things look on a page. Think of it this way: a block element is like a new paragraph—it takes up the full width and always starts on a new line. An inline element, on the other hand, is like a single word within that paragraph; it only takes up as much space as it needs and stays on the same line.

Block Elements

Block elements are the building blocks of a page's layout. They create new content sections and stack vertically, one after another.

<div>

The <div> is a general-purpose container. It's super useful for grouping other elements together when you need to apply styles or structure your page.

This is a simple container. It's a block element, so it takes up the whole line.

<p>

The <p> element is for text paragraphs. It automatically adds space above and below, making your content easy to read.

<blockquote>

<hr>

Inline Elements

Inline elements are for styling and formatting small parts of text. They don't force a new line and can be placed right in the middle of a sentence.

<span>

The <span> is an inline container with no special meaning. It's perfect for applying styling to a specific word or phrase, but we can't show it with the style attribute. It works just like a transparent box around the text.

This is some text with a span element inside it.

<a>

The <a> element is for creating hyperlinks. It's how you link to another page or resource on the web, like this link to a list of HTML elements.

<b>, <i>, <u>, <strong>, <em>

These tags are for styling text. Use <b> for bold, <i> for italics, and <u> for underlines. For more semantic meaning, use <strong> for important text and <em> for emphasized text.

<sup>, <sub>

These are for superscript and subscript text. They are commonly used for formulas like E = mc2 or chemical compounds like H2O.

<mark>

The <mark> element highlights text, making it stand out as relevant. It usually renders with a yellow background, like a digital highlighter.

Please remember to submit the assignment by Friday.

<abbr>

The <abbr> element defines an abbreviation or acronym. The full term is often shown in a tooltip when you hover over it, thanks to the title attribute.

I'm learning about HTML for my course.

<code>

Use the <code> element to display a short piece of code or a programming variable within a sentence, like console.log("Hello, World!").
